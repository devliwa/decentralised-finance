## ðŸ”— 2. Key Features of Web3

- **Decentralization:** Data and apps live on blockchains or peer-to-peer networks, not corporate servers.  
- **Ownership:** Users control their data and digital assets via **crypto wallets** (e.g., MetaMask).  
- **Smart Contracts:** Self-executing agreements that remove intermediaries.  
- **Cryptocurrencies & Tokens:** Used for payments, governance, or rewards.  
- **Identity:** **Decentralized identities (DIDs)** replace logins like Google or Facebook.
